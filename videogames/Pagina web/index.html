Dictionary

HP

Hit-points refers to the life aura a player has when playing the game. Or in simpler terms this resembles the life the character has in game

MP

Mana is a magic based statistic used for when a character decides to use a special attack. This can vary depending on the class the user chooses

STR

Abreviation for strength, and can determine how strong a character's attack is. Strength is a character's physical power. This can be used to deal more damage to enemies and bosses

LCK

Luck is a character's chance of getting a critical hit. This can be used to deal more damage to enemies and bosses

CHR

Charisma is a character's appeal to other characters in the game. It can help the user get discounts in Gandalf's Store as well as recieving better rewards

DEF

Defense is a character's ability to defend against attacks. This can be used to reduce the damage taken from enemies and bosses

ACC

Accuracy is a character's ability to hit an enemy. This can be used to increase the chance of hitting an enemy

AGL

Agility is a character's ability to dodge an enemy's attack. This can be used to increase the chance of dodging an enemy's attack
